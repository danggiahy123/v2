# 📚 Movie App Documentation

> **Tài liệu hướng dẫn cho Movie App - React Native/Expo**

## 📋 **Tổng quan dự án**

Movie App là ứng dụng xem phim được xây dựng với React Native/Expo, bao gồm:
- 🎬 Movie Detail Screen với video player
- 🔍 Search và navigation system
- 🎌 Anime, Series, Movies sections
- 👤 User authentication và interactions
- 📱 Cross-platform mobile app
